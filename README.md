# EBLOG

--------------------
## 项目简介

此项目是基于springboot-vue前后端分离的后端代码，目前在稳定更新中！

## 技术选型

基础框架: Spring Boot + mybatis + log4j + Quartz 

数据缓存:  redis mysql

## 本地构建

启动前需要构建eblog数据库(暂时不支持自动创建)，并且在数据库下面执行sql脚本。

启动时候的需要同时启动mysql和redis，账户密码可以根据配置文件自行选择。
